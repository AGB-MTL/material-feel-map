# The Material Feel Map

An open, shared vocabulary for how materials feel.

**Try the interactive map:** https://agb-mtl.github.io/material-feel-map/material_feel_map.html
*(this link goes live once GitHub Pages is turned on; confirm the exact address in Settings → Pages)*

## What this is

Some senses have precise, shared languages. Color has RGB and Pantone. Sound has pitch and notation dense enough to rebuild a symphony from marks on a page. Touch has nothing like this. Ask ten people how a fabric feels and you get ten vocabularies, with no reliable way to know whether any two of them mean the same thing.

The Material Feel Map is a first attempt at that shared vocabulary. It treats touch as six independent sensory dimensions, texture, give, temperature, moisture, grip, and weight, that combine into a feeling and a meaning. The claim behind it is short: sensation becomes feeling becomes meaning, through combination. Soft and warm read as comforting. Soft and cool, the same softness with a different temperature, read as calm and refined instead.

The map is deliberately made of independent dimensions rather than fixed categories, so it never forces two sensations to travel together when they do not. It marks contested combinations openly (rough can read as natural or as unfinished), and it leaves uncharted combinations blank, because those blanks are where the research is.

## About SENSUS

This is part of SENSUS, a project on the felt meaning of materials. The map is the vocabulary. The next step is an open benchmark that tests which of these mappings actually hold, for whom, and how well machine models can predict them. The essay in this repo lays out the full thinking.
